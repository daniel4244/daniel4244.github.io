# Biblioteca — Sistema de Gerenciamento

Sistema de gerenciamento de biblioteca em Java.

## Conceitos:
- Orientação a Objetos
- Encapsulamento com getters e setters
- Construtores com sobrecarga
- Composição de objetos
- ArrayList e manipulação de listas
- Enum

## Como executar:
Importe o projeto no Eclipse e execute a classe `Main.java`.
