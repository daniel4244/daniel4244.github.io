Desafio 01

Faça um programa que teste a força de uma senha

Regras

Classes de Caracteres:
(minúsculas, maiúsculas, números ou caracteres especiais)

  • Senha fraca: só uma classe de caracteres
  • Senha moderada: possui duas classe de caracteres
  • Senha forte: possui três classe de caracteres
  • Senha muito forte: possui três classe de caracteres

Atividade 02

02. Faça um programa que inverta uma palavra

Exemplo: Azul -> Luza

Atividade 03

03. Faça um programa que marque no texto todas as ocorrencias de um termo digitado

Exemplo: "Amanhã será um novo amanhã, pois amanhã é o dia do amanhã"; Termo: Amanhã, Qtda: 4

Atividade 04

04. Faça um programa que codifique uma frase usando a estratégia TENIS/POLAR, na qual as letras que apareçam em uma das palavras são intercambiadas - A letra T por P; E por O; N por L; S por R e vice-versa. As demais letras devem ser mantidas no texto codificado.

Exemplo: Blusa para o frio -> Bluri tisi e fsao
